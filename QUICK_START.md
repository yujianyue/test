# 快速开始指南

## 5分钟快速部署

### 第一步：配置数据库

编辑 `inc/conn.php` 文件，修改数据库连接信息：

```php
define('DB_HOST', 'localhost');      // 数据库主机
define('DB_USER', 'root');           // 数据库用户名
define('DB_PASS', '');               // 数据库密码
define('DB_NAME', 'quiz_system');    // 数据库名称
```

### 第二步：运行安装程序

在浏览器中访问：
```
http://your-domain/install.php
```

勾选"导入演示数据"，点击"开始安装"按钮。

### 第三步：登录使用

#### 管理员登录
- 访问：`http://your-domain/Ti.php`
- 账号：`admin`
- 密码：`123456`

#### 学生登录
- 访问：`http://your-domain/Da.php`
- 账号：`S0001` 至 `S0036`
- 密码：`123456`

### 第四步：开始答题

学生登录后：
1. 输入考试ID：`demo`（或 `test1`、`survey`）
2. 点击"查询考试"
3. 点击"进入答题"
4. 完成答题后点击"完成答题"

## 常用操作

### 添加新题库

1. 在 `Ku/` 目录创建新的JSON文件，例如 `exam001.json`
2. 编辑文件内容：

```json
{
    "name": "考试名称",
    "mode": 1,
    "time_limit": 60,
    "description": "考试说明",
    "questions": {
        "q1": {
            "type": "single",
            "title": "这是一道单选题？",
            "options": ["选项A", "选项B", "选项C", "选项D"],
            "answer": "A",
            "score": 10
        }
    }
}
```

3. 学生可以使用文件名（不含.json）作为考试ID进行答题

### 批量导入用户

1. 登录管理员后台
2. 点击"用户导入"菜单
3. 按照格式粘贴数据（使用Tab键分隔）：

```
S2001	张三	123456
S2002	李四	abc123
S2003	王五	pass456
```

4. 点击"开始导入"

### 查看答题记录

管理员端：
1. 登录管理员后台
2. 点击"答题记录"菜单
3. 可以按考试名称和学号筛选
4. 点击"详情"查看具体答题情况

### 下载答题数据

1. 登录管理员后台
2. 点击"数据下载"菜单
3. 选择考试名称
4. 点击"下载答题记录"或"下载计分明细"

## 题目类型说明

### 判断题（type: "tf"）
```json
{
    "type": "tf",
    "title": "题目内容",
    "options": ["正确", "错误"],
    "answer": "A",
    "score": 10
}
```

### 单选题（type: "single"）
```json
{
    "type": "single",
    "title": "题目内容",
    "options": ["选项A", "选项B", "选项C", "选项D"],
    "answer": "B",
    "score": 10
}
```

### 多选题（type: "multi"）
```json
{
    "type": "multi",
    "title": "题目内容",
    "options": ["选项A", "选项B", "选项C", "选项D"],
    "answer": ["A", "B", "C"],
    "score": 15
}
```

## 计分模式

在题库JSON中设置 `mode` 参数：

- **mode: 1** - 答题模式1（全对得分）
  - 所有题型都是全对得分，否则0分
  
- **mode: 2** - 调查模式（顺序得分）
  - 单选/判断：A=0, B=1, C=2, D=3...
  - 多选：计算选中个数
  
- **mode: 3** - 答题模式2（混合计分）
  - 单选/判断：全对得分
  - 多选：正确个数 - 错误个数（最低0分）

## 常见问题

### Q: 忘记管理员密码怎么办？
A: 可以在 `inc/json_user.php` 中重置，或重新运行 `install.php`

### Q: 学生答题到一半刷新了页面会丢失吗？
A: 不会，答案保存在Cookie中，刷新后会自动恢复

### Q: 可以设置答题时间限制吗？
A: 可以，在题库JSON中设置 `time_limit` 参数（单位：分钟）

### Q: 如何查看学生的详细答题情况？
A: 管理员后台→答题记录→点击"详情"按钮

### Q: 数据如何备份？
A: 定期导出MySQL数据库即可
